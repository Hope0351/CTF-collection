# Learn SQL injection in practice by hacking vulnerable application! — StackZero


## Introduction


Trying some SQL injection attacks in practice, in my opinion, is the best way to understand that.
Just searching online, I found that setting up a laboratory can be very long and tricky, especially for a beginner, and it often has the consequence of causing the student to give up.
What I did to make the process simpler, is to build a terrible coded web application in python named [SDWA](https://github.com/StackZeroSec/sdwa) (Sitting Duck Vulnerable App) .
It’s just in an embryonic phase, but we can use it to learn:


- XSS

- SQL injection

- Command injection


I have already talked about the XSS part in [this article](https://medium.com/@stackzero/the-terrifying-world-of-cross-site-scripting-xss-part-2-stackzero-cc7fa7e8dcbb), so it’s time to focus on SQL injection and put our knowledge into practice.
If you have no idea of what it is, I suggest you take your time and read [this introduction](https://medium.com/codex/sql-injection-what-you-need-to-know-stackzero-abc80bc1ea5e).
