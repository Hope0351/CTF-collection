## Methodology

This corpus was compiled via 86 web-search queries across 8 thematic axes (awesome-lists, per-category, per-event, per-team, per-author, per-platform, the collection topics, year-range). Each query returned up to 20 results. After de-duplication, 376 unique sources were identified across 5 platforms (the collection, Medium, YouTube, CTFtime, and personal blogs).

For each the collection source, the the collection API was queried (authenticated) for full metadata: stars, forks, primary language, last-push date, topics, license, and description. For non-the collection sources, the search snippet and title were used for categorization.

Categories were detected via keyword matching against titles and snippets. CTF events were detected via a curated pattern list of 50+ known events. Years were extracted via regex matching. Difficulty was assessed heuristically based on the presence of tier-indicating keywords (beginner, picoCTF → Beginner; DEF CON, hxp → Elite; etc.).

For full methodology details, see [`methodology.md`](methodology.md).
