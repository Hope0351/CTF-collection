## ­¤Ä» Suggested Learning Order

If you're new to web exploitation, work through the resources in this order:

1. **Start with a structured learning platform.** Begin with [PortSwigger Web Security Academy](https://portswigger.net/web-security) ŌĆö work through all Apprentice and Practitioner labs. Then read [`orangetw/My-CTF-Web-Challenges`](../02-web/archives/My-CTF-Web-Challenges) to see how top-tier web challenges are designed.

2. **Build a personal writeup habit.** Every time you solve a challenge, write a 2-paragraph summary. Commit these to a personal the collection repository ŌĆö over time, this becomes your portfolio.
3. **Study elite writeups.** After each major CTF, read the top-placing teams' writeups within 1 week. The team archives at `p4-team/ctf`, `r3kapig/writeup`, and `theori-io/ctf` are the strongest references.
4. **Play live.** Join [CTFtime](https://ctftime.org/) and play at least one CTF per month. There is no substitute for live competition.
