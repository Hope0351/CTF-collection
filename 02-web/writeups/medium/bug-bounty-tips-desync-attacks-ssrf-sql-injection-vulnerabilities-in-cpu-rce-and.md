## 📝 5 Infosec Articles

#1 Learn how to combine cross-domain requests with server flaws to poison browser connection pools, install backdoors, and release desync worms through [browser powered desync attacks](https://portswigger.net/research/browser-powered-desync-attacks). by [James Kettle](https://twitter.com/albinowax).#2 [Enhancing the subdomain enumeration by including a special DNS node](https://www.securesystems.de/blog/enhancing-subdomain-enumeration-ents-and-noerror/) that is often ignored — ENTs and NOERROR.#3 [Bypassing authentication when accessing the same endpoint (mboximport) used by CVE-2022–27925](https://www.volexity.com/blog/2022/08/10/mass-exploitation-of-unauthenticated-zimbra-rce-cve-2022-27925/) in [Zimbra Collaboration Suite](https://www.zimbra.com/downloads/zimbra-collaboration/) (ZCS) email servers, making the vulnerability significantly more critical in severity.#4 @Vimanari shares how they found [an unauthenticated Jenkins dashboard that belonged to a Fortune 500 company from Shodan](https://systemweakness.com/rooting-jenkins-remote-code-execution-on-a-live-bug-bounty-target-fc2c12d89a2e) resulting in RCE.#5 [The quantum state of Linux kernel garbage collection CVE-2021–0920 (Part I)](https://googleprojectzero.blogspot.com/2022/08/the-quantum-state-of-linux-kernel.html?m=1) is a deep dive into an in-the-wild Android exploit where a surveillance vendor named Wintego was able to remotely root Samsung devices.
