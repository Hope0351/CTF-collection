# How I hacked Nepal’s ecommerce site and bought huge packs of kitkats for free lol :) | subhdhungana

Before few months when I was tired of doing my college assignments, I started to drill up my skills in ethical hacking. I used to practise testing the web application sites too.

Then I thought, it’d be the great idea if I’d practice web application testing by testing on the real ecommerce site of Nepal.

Then I started searching on google, top ecommerce sites of Nepal and chosen few lists. I knew which bug I’d test first i.e parameter tampering bug because it’s so exciting to test as I was starting to learn about testing, bug bounties, etc. at that time. So, it was new for me at that time which attracted me to test parameter tampering bug. I used to read about bug articles on medium.

I found one article writtten by Vedant if I’m not wrong. He wrote how he hacked the newspaper subscription site and manipulated the price. As a beginner at that time, I thought this has to be tested. This bug is interesting.

After testing for few minutes, I suddenly found paramter tampering bugs on many sites. Out of which I started to exploit one higher ranked ecommerce site to test whether it will work or not.

Lets keep the name of that ecommerce site private and name it redacated.com for now.

Point to be noted here, I already reported the bug on the redacated.com site. I notified them about the bug via email and other communication sources.

After reporting the bug to the site developers and administration. I thought, let’s give it a try to test whether this is real or not!

Since, I liked a Kitkat a lot. I thought of ordering many packs kitkats and other sets of chocolates just to give it a try. I didn’t ordered expensive stuffs to put company in trouble as I already reported that bug to the company and made them aware about my testing.

I intercepted the network of that redacated.com site using Burp Suite. After that, I started to play with the basic pricing and quantities parameter using burp suite after intercepting the network. I somehow manipulated the quantities parameter in such a way that it perfectly fits within my desired amount. I still remember I assigned the targeted parameter value to Rs.10 and payed using payment app. It was successfully done.

So, I found the parameter tampering bug in redacated.com site which means I could easily manipulate the items price. I could easily buy Rs. 1 lakh laptop at my desired rate unless they knew about it. At that time, I thought this was cool. I’ve never done such kind of stuffs since that time. So I thought to give it a try.

After few days, the order was successfull. It was delivered!!!!!!!!

After that, I reported to the company again about the bug.

I think, now the bug has been fixed!
