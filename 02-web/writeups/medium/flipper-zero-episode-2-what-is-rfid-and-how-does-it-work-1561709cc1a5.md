## How Flipper Zero Works with RFID

Flipper Zero’s RFID module allows you to:

- Read RFID tags and analyze their data

- Emulate RFID tags to act as an existing card

- Write RFID data to compatible blank tags

These features make it useful for penetration testers, security researchers, and those interested in understanding RFID security.

Now, let’s go through how to use Flipper Zero for reading, emulating, and writing RFID tags.

### Reading an RFID Card with Flipper Zero

- Open the RFID menu on Flipper Zero.

2. Place the RFID tag against the back of Flipper Zero.

*Reading RFID tag with Flipper Zero*

3. It will display the tag’s details.

### Emulating an RFID Tag

Once you’ve read an RFID tag, Flipper Zero can emulate it, acting like the original tag.

- Select the saved RFID tag from the device’s memory.

- Tap “Emulate” to activate the emulation mode.

- Hold Flipper Zero near an RFID reader to test the emulation.

*Emulating RFID tag with Flipper Zero*

Some access control systems have additional security measures that prevent simple cloning, so emulation may not always work.

### Writing an RFID Tag

If you have a blank writable RFID tag, you can use Flipper Zero to copy and write data onto it.

- Select the saved RFID tag from Flipper Zero’s memory.

2. Place a blank RFID tag on the back of the device.

3. Tap “Write” to transfer the data to the new tag.

*Writing RFID card using Flipper Zero*

Writing is useful for creating backups of RFID access cards and testing security implementations.
