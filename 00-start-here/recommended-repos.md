## How to use these

- **For browsing:** Start with `sajjadium/ctf-archives` — it's the closest thing to a complete CTF challenge corpus.
- **For learning paths:** Start with `apsdehal/awesome-ctf` for the directory, then drill into `pwn.college` for pwn and `picoCTF` for general beginner content.
- **For web specifically:** `orangetw/My-CTF-Web-Challenges` is the gold standard.
- **For blockchain:** `minaminao/ctf-blockchain` (200+ challenges, Foundry-based).
- **For HackTheBox specifically:** `momenbasel/htb-writeups` (500+ machines/challenges with interactive knowledge graph).
- **For elite-tier reference:** `p4-team/ctf` and `r3kapig/writeup` cover DEF CON, hxp, 0CTF at competition-level depth.
- **For official challenge source:** `google/google-ctf` (Google CTF since 2017) and the Nautilus Institute repos (DEF CON CTF 2024+).
