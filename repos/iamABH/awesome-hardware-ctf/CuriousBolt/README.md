# Curious Bolt 

## Description

The CuriousBolt is a Hardware Hacking multitool that functions as a 
crowbar voltage glitcher, an 8-channel logic analyzer, and differential power scope. 

It is provided with a STM32F103 target board that includes the 4 hardware security challenges of ECSC23
